O que está incluído

Funcionalidades

Tudo que foi desenvolvido para o site da Forcemak Prime.

🎬

Animações Profissionais

GSAP para o hero com entrada sequencial. AOS para animações ao rolar. Contadores numéricos. Transições suaves entre páginas. Cards com hover effect.

📦

Busca + Estoque em Tempo Real

Sidebar com busca por nome/descrição, filtros por categoria e toggle de disponibilidade. Badges 🟢 disponível / 🟡 últimas unidades / 🔴 indisponível.

📱

Mídias Sociais Incorporadas

YouTube, Instagram e Facebook embarcados dentro do site. O lead vem da rede para o site — não vai embora. Configurável pelo admin sem código.

🔐

Painel Admin Completo

Login com JWT. Gestão de produtos, estoque, textos do site, mensagens. Sem precisar de programador para alterar conteúdo.

📊

Facebook Pixel + API

Pixel ID 1433120138425099 ativo em todas as páginas. Rastreio de PageView, Contact e ViewContent. Métricas visíveis no painel admin.

🔍

SEO Completo

Meta tags, Open Graph (preview no WhatsApp), JSON-LD Schema.org, sitemap.xml, robots.txt, canonical URLs e breadcrumbs estruturados.

📱

100% Responsivo

Layout adaptável para celular, tablet e desktop. Menu hambúrguer no mobile. Grid flexível que reorganiza os elementos conforme a tela.

⚡

Performance

Imagens com lazy loading. Scripts carregados ao final. Fontes com preconnect. CSS e JS organizados em arquivos separados por contexto.

💬

WhatsApp Integrado

Botão fixo de WhatsApp em todas as páginas com mensagem pré-preenchida. Clique no produto envia o nome do equipamento diretamente no chat.
